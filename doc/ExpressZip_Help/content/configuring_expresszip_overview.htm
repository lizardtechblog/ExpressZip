Configuring ExpressZip Overview

You can configure ExpressZip to complete the following tasks:

  • Change the export directory.
  • Add a custom base layer.
  • Change the default base layer.
  • Change the Express Server WMS host.
  • Enable mail notifications.

The main ExpressZip configuration file is stored in the following location:

<Installation Directory>\ImageServer\Tomcat\conf\ExpressZip.properties

NOTE: When you change the configuration of ExpressZip, restart Tomcat for the changes to take effect.
