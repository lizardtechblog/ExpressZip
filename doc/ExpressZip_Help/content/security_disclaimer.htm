Security Disclaimer

For ease of use and for simple extensibility, the ExpressZip application does not require user authentication. If your instance of the ExpressZip application is accessible over the internet or a public network, you should be aware of the following potential security concerns:

  • Any image export jobs that you submit are accessible to all of the users of the application. This may be an issue if the region of interest that you want to export is of a sensitive nature.
  • Any user can access the Job Manager to remove current and existing jobs, even jobs submitted by other users.
