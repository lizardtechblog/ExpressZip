Accessing ExpressZip

By default, you can access ExpressZip via HTTPS and HTTP. Both are protocols for communication over a network, however, HTTPS adds encryption for an increased layer of security.

To access ExpressZip, open a web browser and navigate to one of the following URLs:

  • For HTTP:

    http://<Express Server name>:8080/ExpressZip

  • For HTTPS:

    https://<Express Server name>:<https port>/ExpressZip

The HTTPS port number is configured during installation. By default, the port number for HTTPS is 8443 and the HTTP port number is 8080. For example, to access ExpressZip via HTTPS, you might enter the following URL:

https://local-express-server:8443/ExpressZip
